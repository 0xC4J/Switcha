Switcha Proxy Switcher does not collect, transmit, sell, or share any user data. Data Storage - Proxy profiles (proxy type, host, port, and profile name) are stored locally in the user's browser using chrome.storage.local. - The extension does not sync data to any external server. Network / Tracking - The extension does not include analytics, trackers, ads, or remote code. - The extension does not send any data to third parties. Permissions - proxy: required to apply the proxy configuration selected by the user. - storage: required to save proxy profiles locally. - tabs: used only to identify the active tab domain and populate the domain dropdown in the popup UI. - host permissions (): required to allow applying proxy settings globally or for user-selected domains. - The extension does not inject scripts into web pages and does not read or modify website content.
